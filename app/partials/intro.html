ng-drawing Web app that draws shapes on a page based on a user's commands

Commands available:

  • clear 'colour': set the whole canvas to the given colour
  • rectangle 'x' 'y' 'w' 'h' 'colour': draw an unfilled rectangle with a border of the given colour
  • line 'x1' 'y1' 'x2' 'y2' 'colour': draw a line with the given colour
  • fill 'x' 'y' 'colour': flood-fill starting at the given location
